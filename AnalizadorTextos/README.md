# Proyecto Final: Analizador y Transformador de Textos

Este repositorio contiene el proyecto final de la asignatura, enfocado en la creación de una herramienta para el análisis, manipulación y transformación de ficheros de texto.

## Referencia
Este proyecto se ha realizado siguiendo las especificaciones detalladas en el enunciado oficial: **Proyecto Final: Analizador y Transformador de Textos**.

El sistema permite la limpieza de datos, cálculo de estadísticas de frecuencia, detección de anagramas mediante recursividad y la aplicación de algoritmos de cifrado César.
